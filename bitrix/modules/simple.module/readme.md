# Тестовый модуль 

## Описание
Модуль для платформы 1С-Битрикс

## Структура компонента
simple.module/

├── admin/─────────────────────────────────── Административные файлы  
├── install/─────────────────────────────────── Установочные файлы  
│   ├── db/──────────────────────────────────── Миграции БД  
│   │   ├── mysql/install.sql  
│   │   └── postgresql/install.sql  
│   ├── components/────────────────────────────── Компоненты модуля  
│   ├── version.php─────────────────────────────── Версия модуля  
│   └── index.php──────────────────────────────── Класс установки  
├── lib/───────────────────────────────────── Основная бизнес-логика  
│   ├── orm/─────────────────────────────────── ORM сущности  
│   │   └── exampletable.php  
│   └── manager.php───────────────────────────── Менеджер модуля  
├── include.php──────────────────────────────── Главный файл модуля  
└── default_option.php─────────────────────────── Настройки по умолчанию
